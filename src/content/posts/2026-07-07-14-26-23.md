## []([https://linux.do/t/topic/2300796#p-18613039-h-22-9)2.2](https://linux.do/t/topic/2300796#p-18613039-h-22-9)2.2) 创建机器

**点击1，复制下面代码运行，创建韩国/日本/香港机器**

韩国：

```sql

az group create --name proxy-korea-group --location koreacentral && \

az network public-ip create --resource-group proxy-korea-group --name proxy-korea-ip --location koreacentral --sku Standard --allocation-method Static && \

az vm create --resource-group proxy-korea-group --name proxy-korea-vm --location koreacentral --image Canonical:ubuntu-24_04-lts:server:latest --size Standard_B2ats_v2 --admin-username azureuser --generate-ssh-keys --public-ip-address proxy-korea-ip --os-disk-size-gb 64 --storage-sku Standard_LRS && \

az vm open-port --resource-group proxy-korea-group --name proxy-korea-vm --port 80 --priority 100 && \

az vm open-port --resource-group proxy-korea-group --name proxy-korea-vm --port 443 --priority 110 && \

az vm open-port --resource-group proxy-korea-group --name proxy-korea-vm --port 54321 --priority 120

```

日本：

```sql

az group create --name proxy-jpw-group --location japanwest && \

az network public-ip create --resource-group proxy-jpw-group --name proxy-jpw-ip --location japanwest --sku Standard --allocation-method Static && \

az vm create --resource-group proxy-jpw-group --name proxy-jpw-vm --location japanwest --image Canonical:ubuntu-24_04-lts:server:latest --size Standard_B2ats_v2 --admin-username azureuser --generate-ssh-keys --public-ip-address proxy-jpw-ip --os-disk-size-gb 64 --storage-sku Standard_LRS && \

az vm open-port --resource-group proxy-jpw-group --name proxy-jpw-vm --port 80 --priority 100 && \

az vm open-port --resource-group proxy-jpw-group --name proxy-jpw-vm --port 443 --priority 110 && \

az vm open-port --resource-group proxy-jpw-group --name proxy-jpw-vm --port 54321 --priority 120

```

香港：

```sql

az group create --name proxy-hk-group --location eastasia && \
```
